# Memory-Matching-Game
A simple Memory Matching Game built with HTML, CSS, and JavaScript (jQuery). The game challenges players to match pairs of emojis in the fewest moves and shortest time possible.

# Features

16 cards (8 pairs of emojis) randomly shuffled every game

Track number of moves and time taken

Visual feedback for matched and unmatched cards

Sound effects for correct and wrong matches

Winning message displayed when all pairs are matched

Fully responsive and interactive

# Technologies Used

HTML5 – for page structure

CSS3 – for styling and animations

JavaScript (jQuery) – for game logic and interactivity

Audio – match and wrong sound effects
